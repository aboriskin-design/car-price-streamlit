# HW1_ML_Boriskin: car price

что сделано:
1) провел EDA: посмотрел распределения числовых признаков, сравнил train и test, посмотрел связи на pairplot и по корреляциям.  
2) сделал предобработку: почистил пропуски, привел форматы, для name сделал brand.  
3) обучил модели:
LinearRegression на числах
Lasso, ElasticNet с подбором параметров через GridSearchCV
Ridge на cat+num после one-hot и подбор alpha через GridSearchCV

лучше всего по качеству (R2) показала себя Ridge на cat+num после one-hot.  
R2 test получилось около 0.77, MSE test около 1.3e10.
самый большой прирост дало добавление категориальных признаков и one-hot кодирование, после этого Ridge стал получше базовой линейной регрессии.

что не удалось и почему:
часть моделей не занулили коэффициенты, тк оптимальные параметры регуляризации получились мягкими и признаков не так много.

Streamlit:
сделал приложение на Streamlit:
вкладка EDA: строит простые гистограммы по загруженному csv
вкладка Predict: можно загрузить csv и получить предсказания или ввести признаки вручную
вкладка Model weights: показывает топ коэффициентов модели

ограничения: если в csv совсем другие колонки или странные значения, могут быть ошибки, поэтому лучше использовать те же признаки, что были в обучении.
